h1. Memverse

A Bible memorization application

"!https://app.wercker.com/status/a6bf54c9eb901816e1f6aee1ae3ad20a/m/(wercker status)!":https://app.wercker.com/project/bykey/a6bf54c9eb901816e1f6aee1ae3ad20a

h2. Getting Started

* Clone this repository.
* Start the containers: @docker-compose up@
* Create and seed the database: @docker-compose run web bundle exec rake db:setup@

h2. Dependencies

To use this application, you will need:

* Docker Compose

h2. Running tests

Unit tests: @bundle exec rspec@
Integration tests: @bundle exec cucumber features@
Javascript tests: @bundle exec rake spec:javascript RAILS_ENV=test@
